Projeto Será que é Autismo?

O projeto "Será que é Autismo?" foi criado por Vitor Hugo Ribeiro Sá no CEDUP Renato Ramos da Silva. A proposta é oferecer uma ferramenta gratuita e acessível para uma triagem inicial de sinais relacionados ao Transtorno do Espectro Autista (TEA).

O site apresenta um questionário inspirado em modelos de triagem, além de informações sobre autismo, perguntas frequentes e orientações para incentivar a busca por avaliação profissional. O projeto não substitui diagnóstico médico, mas ajuda famílias a entenderem melhor quando procurar apoio especializado.

Desenvolvido com HTML, CSS e JavaScript, o projeto também representa uma etapa importante da minha formação em tecnologia e do meu aprendizado em desenvolvimento web.
